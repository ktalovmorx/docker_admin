<h3> Resumen </h3>

```
    Permite gestionar de manera gráfica las instancias de dockers
```

<h3> Instalación </h3>

```
sudo apt install python3-pip
sudo apt install python3.11-venv
cd /root
python3 -m venv env3
source env3/bin/activate
pip3 install flask docker
mkdir docker_admin
```
